# تطبيق قائمة المهام

هذا تطبيق ويب تقدمي (PWA) بسيط يتيح إدارة المهام اليومية. يمكنك إضافة المهام وحذفها، وستظل محفوظة على جهازك حتى بعد إغلاق المتصفح.

## المميزات

- إضافة مهام جديدة
- حذف المهام المكتملة
- حفظ البيانات محليًا
- واجهة بسيطة وسهلة الاستخدام
- يعمل على جميع الأجهزة

## التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript
- تقنيات تطبيقات الويب التقدمية (PWA)
